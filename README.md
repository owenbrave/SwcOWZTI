## 前言

家校合作平台是一个以促进家庭与学校之间沟通、合作，共同关注和参与孩子成长为核心目的的教育信息管理系统。该平台通过整合学校教育资源、家庭关注重点，为用户提供了一个便捷、高效的家校沟通平台。

## 内容介绍

家校合作平台旨在建立一个稳定、可靠的家校沟通桥梁，通过实时、便捷的信息交互，让家长能及时了解孩子的学习和生活情况，参与学校教育，同时，也让学校能够更全面地了解学生的家庭背景，实现家校共育。

## 技术介绍

语言：Java
使用框架：Spring Boot
前端技术：JS、Vue、css3
开发工具：IDEA/Eclipse
数据库：MySQL 5.7/8.0
数据库管理工具：phpstudy/Navicat
JDK版本：jdk1.8
Maven: apache-maven 3.8.1-bin
前端环境：Node.js 12\14\16
